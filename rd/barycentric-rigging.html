Barycentric Rigging

MayaSkin Weightsgithub.com/pecktd/barycentric_skin_transfer

Check out the code in this repository to see how I applied barycentric coordinates using an algebraic approach to extract skin weight values.

I've understood the concept of barycentric coordinates for a while, mainly from a geometric perspective. However, that approach is quite difficult to implement using Maya's utility nodes.

Recently I discovered that the algebraic approach is not only more optimised but also much easier to integrate within Maya's node-based system.
