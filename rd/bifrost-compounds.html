Blend Shape with Custom IDs

The setup and example can be found here: gitlab.com/pecktd/id_based_blend_shape_bif

It uses a multi (array) attribute as input so users can define the target vertex IDs for each base mesh vertex.

In script.py there is a function get_target_order to help users get an array of target vertex IDs that share the same position with the base mesh vertex IDs.

I've been experimenting to create a blend shape node that allows arbitrary target vertex IDs. I've done it with Maya Python API and found the speed is similar to using wrap / blend shape method.

Big thanks to this excellent tutorial from Marco D'Ambros.

Blend Shape with 3-Points Curve

Continuing experiments on Bifrost. This time I applied a Quadratic Spline Function to create a curvature of the vertex transformation.

The vector from the given pivot point to the middle position between base and target defines the middle point (bulge) of the spline curve.

The example scene and Bifrost graph can be found here: gitlab.com/pecktd/quadratic_spline_blend_shape
