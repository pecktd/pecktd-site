Blend Shape Exporter

MayaToolBlend Shape

A toolset to store / restore blend shape data.

The data is kept as the structure below:

{
    'name': 'blendShape1',
    'weight_attrs': ['nurbsSphereShape1', 'nurbsSphereShape2'],
    'weights': [1.00, 0.00],
    'drivers': ['loc1.driver', None],
    'output_geos': ['nurbsSphereShape1', 'nurbsSphereShape2'],
    'input_targets': [    # List of geos.
        [    # Geos[0]
            {    # Shape[0]
                'input_geo_target': 'nurbsSphereShape1.worldSpace[0]',
                'input_points_target': [(0.00, 0.00, 0.00, 1.00), ...],
                'input_components_target': ['vtx[3:4]', 'vtx[6]', ...],
                'inbetweens': {
                    5500: {    # Inbetweens[0.5]
                        'input_geo_target': None,
                        'input_points_target': [(0.00, 0.00, 0.00, 1.00), ...],
                        'input_components_target': ['vtx[3:4]', 'vtx[6]', ...]
                    }
                }
            },
            {    # Shape[1]
                'input_geo_target': None,
                'input_points_target': [(0.00, 0.00, 0.00, 1.00), ...],
                'input_components_target': ['vtx[3:4]', 'vtx[6]', ...]
            }
        ],
        [    # Geos[1]
            { },    # Shape[0]
            { }     # Shape[1]
        ]
    ]
}
