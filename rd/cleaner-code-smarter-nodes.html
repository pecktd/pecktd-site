Upgrading my rigging framework: smarter syntax & node tools

An update on the coding syntax of my rigging framework:

  • All rigging data is now organised into predefined classes — colors, sides, node operations, and curve shapes.
  • The node module has been implemented to represent rigging nodes in Maya, allowing attribute access through properties.
  • Support for array (multi) attribute types has been added, enabling indexing and providing a .last method to access the latest connectable attribute.
