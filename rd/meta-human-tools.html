Meta Human Tools

MetaHumanFACSPython

The demo of my facial rig uses FACS shapes extracted from the MetaHuman rig.

Toolset for modifying MetaHuman DNA files & extracting FACS shapes

Here is a toolset to modify / update the DNA file from MetaHuman.

I've been studying FACS and found that MetaHuman is a great source for learning. So, I developed a tool to work with the DNA files from Epic Games to extract the FACS shapes and build them with my custom rigs.

Testing build process & performance

  • Build time: 26.04 seconds for the facial rig with 71 key shapes, 20 corrective shapes, and 26 tweakers.
  • Build time: 15.07 seconds for the body part and assembling the facial rig to the body part.
  • Average FPS: 40–50.
