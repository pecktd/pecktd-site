strik — a Ribbon-IK alternative using weighted matrices

MayaIKModule

strik stands for String IK.

It is an optimised version of Ribbon IK — no NURBS curve or surface needed to simulate the positions.

With the base functions from cyberiRex's gist I use them to calculate the weight values of the input matrices.
